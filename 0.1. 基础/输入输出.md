## 输入多组连续数据

```C
int i;
while(scanf("%d", &i) != EOF){}
```

```C
// 以字符串读入：
char s[300];
while(gets(s)){}
```

## 输入固定位数的整数

```C
// 读取如：20160312 的日期
scanf("%4d%2d%2d", &y, &m, &d);
```

可先保存至一个数组（缓冲区）中；

```C++

```
